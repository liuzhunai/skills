## rental-search

**租房搜索 Skill**

智能搜索多平台个人房源，支持地点关键词、距离过滤、时间过滤。

### 支持平台

| 平台 | 数据类型 | 状态 |
|------|----------|------|
| 58同城 | 个人房源 | ✅ 已支持 |
| 链家 | 待开发 | 🔄 规划中 |
| 自如 | 待开发 | 🔄 规划中 |
| 豆瓣租房 | 待开发 | 🔄 规划中 |

### 快速使用

```
/rental-search 百度科技园附近5km的房源
```

### 主要功能

- 自然语言解析查询参数
- 地理定位与距离过滤
- 多平台房源聚合（规划中）
- 自动生成Excel报告（含图片）

### 架构设计

采用模块化架构，应用多种设计模式：

| 模式 | 应用场景 | 说明 |
|------|----------|------|
| 责任链模式 | 查询参数解析 | 每个解析器处理一种参数类型 |
| 策略模式 | 平台适配器 | 不同平台实现各自的搜索策略 |
| 工厂模式 | 平台创建 | 统一创建平台适配器 |
| 单例模式 | 配置管理 | 全局唯一配置实例 |

## 开发规范

### 新增技能

1. 在本目录下创建技能文件夹（小写字母，连字符分隔）
2. 必须包含 `SKILL.md` - 技能定义文件
3. 建议包含 `README.md` - 使用说明
4. 脚本放在 `scripts/` 目录下，采用模块化设计

### 模块化设计原则

1. **单一职责**: 每个模块只负责一个功能
2. **开放封闭**: 对扩展开放，对修改封闭
3. **依赖倒置**: 依赖抽象而非具体实现
4. **接口隔离**: 使用小而专一的接口

### 设计模式应用

- **责任链模式**: 用于多步骤处理流程（如参数解析）
- **策略模式**: 用于可替换的算法实现（如平台适配）
- **工厂模式**: 用于对象创建的统一管理
- **单例模式**: 用于全局配置或共享资源

### 文档规范

- `SKILL.md`: 触发词、工作流程、技术细节、架构设计
- `README.md`: 快速开始、参数说明、示例输出
