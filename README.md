## rental-search

**租房搜索 Skill**

智能搜索多平台个人房源，支持地点关键词、距离过滤、时间过滤。

### 支持平台

| 平台 | 数据类型 | 状态 |
|------|----------|------|
| 58同城 | 个人房源 | ✅ 已支持 |
| 链家 | 待开发 | 🔄 规划中 |
| 自如 | 待开发 | 🔄 规划中 |
| 豆瓣租房 | 待开发 | 🔄 规划中 |

### 快速使用

```
/rental-search 百度科技园附近5km的房源
```

### 主要功能

- 自然语言解析查询参数
- 地理定位与距离过滤
- 多平台房源聚合（规划中）
- 自动生成Excel报告（含图片）

## 开发规范

### 新增技能

1. 在本目录下创建技能文件夹（小写字母，连字符分隔）
2. 必须包含 `SKILL.md` - 技能定义文件
3. 建议包含 `README.md` - 使用说明
4. 脚本放在 `scripts/` 目录下

### 文档规范

- `SKILL.md`: 触发词、工作流程、技术细节
- `README.md`: 快速开始、参数说明、示例输出
