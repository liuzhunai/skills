# 租房搜索 Skill

智能搜索多平台个人房源，支持地点关键词、距离过滤、时间过滤。目前已支持58同城，后续可拓展其他房源平台。

## 支持平台

| 平台 | 数据类型 | 状态 |
|------|----------|------|
| 58同城 | 个人房源 | ✅ 已支持 |
| 链家 | 待开发 | 🔄 规划中 |
| 自如 | 待开发 | 🔄 规划中 |
| 豆瓣租房 | 待开发 | 🔄 规划中 |

## 快速开始

### 基本用法

```
/rental-search 百度科技园附近5km的房源
```

### 支持的查询格式

| 查询示例 | 解析结果 |
|----------|----------|
| `百度科技园附近5km的房源` | 地点=百度科技园, 距离=5km, 时间=15天 |
| `中关村附近3km 30天内` | 地点=中关村, 距离=3km, 时间=30天 |
| `国贸附近 价格3000-5000` | 地点=国贸, 价格=3000-5000元 |
| `西二旗附近 2000以内` | 地点=西二旗, 价格上限=2000元 |
| `望京附近2km 一居室` | 地点=望京, 距离=2km, 户型=一居 |
| `上地附近 面积30-50平` | 地点=上地, 面积=30-50㎡ |
| `西二旗附近 50平以内的合租` | 地点=西二旗, 面积上限=50㎡, 租赁方式=合租 |

## 参数说明

### 距离参数

- 支持 `km`、`公里`、`米`、`m` 单位
- 默认值: 5km
- 示例: `3km`、`500米`、`2公里`

### 时间参数

- 单位: 天
- 默认值: 15天
- 示例: `7天内`、`30天`、`60天内`

### 价格参数

- 格式1: `价格3000-5000` (区间)
- 格式2: `3000以内` (上限)
- 默认: 不限

### 面积参数

- 格式1: `面积30-50` 或 `30-50平` (区间)
- 格式2: `50平以内` 或 `50平米以下` (上限)
- 默认: 不限

### 户型参数

- `一居` / `一室` / `开间` - 一居室
- `两居` / `两室` - 两居室
- `三居` / `三室` - 三居室
- 默认: 不限

### 租赁方式参数

- `整租` - 整套出租
- `合租` - 合租/单间
- 默认: 不限（搜索所有类型）


## 注意事项

1. **请求频率**: 房源平台有反爬机制，搜索间隔约2-3秒
2. **距离计算**: 使用直线距离，实际通勤距离可能不同
3. **数据时效**: 部分房源可能已下架，建议优先查看最新房源
4. **区域限制**: 目前仅支持北京地区
5. **数据源**: 目前仅搜索58同城个人房源，后续将拓展更多平台

## 扩展开发

### 添加新平台

1. 在 `scripts/` 目录下创建平台适配器
2. 实现 `BasePlatform` 接口:
   - `build_search_url()` - 构建搜索URL
   - `parse_list_page()` - 解析列表页
   - `parse_detail_page()` - 解析详情页
3. 在 `SKILL.md` 中更新支持平台状态
